# Veriflow Pro - User Guide

## Welcome to Veriflow Pro

Veriflow Pro is a professional audio/video workflow application designed for sound engineers, video editors, and post-production professionals.

## Main Features

### 1. OFFLOAD (F1)
Secure file offloading with checksum verification.

**Features:**
- Source and dual destination support
- MD5 checksum verification
- Progress tracking with speed and ETA
- Automatic logging

**Usage:**
1. Select source folder
2. Select destination folder(s)
3. Click "START OFFLOAD"
4. Monitor progress

### 2. MEDIA (F2)
Browse and preview media files.

**Features:**
- File browser with metadata display
- Audio/Video preview
- Quick file information
- Send to other modules

**Usage:**
- Browse folders to view media files
- Click files to see metadata
- Use action buttons to open in Player or send to Transcode

### 3. PLAYER (F3)
Professional audio/video playback.

**Features:**
- LibVLC-powered playback
- Transport controls (Play/Pause/Stop)
- Timeline scrubbing
- Timecode display
- Metadata viewing

**Usage:**
1. Click "OPEN FILE" to load media
2. Use transport controls for playback
3. Scrub timeline for navigation

### 4. SYNC (F4)
Synchronize video and audio files.

**Features:**
- Dual file selection (Video + Audio)
- Waveform comparison
- Sync offset adjustment
- Auto-sync capability

**Usage:**
1. Import video file
2. Import audio file
3. Use Auto Sync or manual offset
4. Apply sync

### 5. TRANSCODE (F5)
Professional media transcoding.

**Features:**
- Multiple format support (ProRes, H.264, DNxHD, etc.)
- Batch processing
- Quality presets
- Progress tracking

**Usage:**
1. Add files to queue
2. Select output format
3. Configure settings
4. Start transcode

### 6. REPORTS (F6)
Generate professional camera and sound reports.

**Features:**
- Camera Report (Video mode)
- Sound Report (Audio mode)
- Clip logging
- PDF export
- EDL/ALE export

**Usage:**
1. Add clips to report
2. Fill in metadata
3. Export to PDF or EDL/ALE

## Keyboard Shortcuts

- **F1** - Offload
- **F2** - Media
- **F3** - Player
- **F4** - Sync
- **F5** - Transcode
- **F6** - Reports

## Getting Started

1. Launch Veriflow Pro
2. Use F1-F6 or bottom navigation to switch modules
3. Each module is designed for specific workflows
4. All modules work together seamlessly
